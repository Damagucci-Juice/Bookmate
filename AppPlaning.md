# Bookmate — 앱 기획 문서

> 책에서 발견한 문장을 스캔하고, 꾸미고, 나만의 컬렉션으로 쌓아가는 독서 기록 앱

---

## 1. 앱 개요

| 항목 | 내용 |
|---|---|
| 앱 이름 | Bookmate |
| 플랫폼 | iOS (iPhone) |
| 프레임워크 | UIKit |
| 핵심 가치 | 책 속 문장을 카메라로 바로 수집 → 카드로 꾸미기 → 태그로 정리 |

## 3. 사용자 흐름 (User Flow)

```
[탭바: 홈]                          [탭바: 내 문장]
    │                                     │
    ▼                                     ▼
1 홈                               7 내 문장 ──── 7-2 태그 필터
    │ (스캔 시작)                         ▲
    ▼                                     │ (저장 완료)
2 도서 선택                               │
    │                               6 문장 수집
    ▼                                 │       │
3-1 사진 촬영                          │    6-2 태그 추가 (Bottom Sheet)
    │ (셔터)                           │
    ▼                                  │
3-2 사진 확인 ──── (다시 촬영) ────► 3-1
    │ (텍스트 인식)
    ▼
3-3 텍스트 인식
    │ (텍스트 확인하기)
    ▼
4 문장 선택 (줄 단위 tap, 최대 5줄)
    │ (계속)
    ▼
5 카드 꾸미기 ──── (스타일 선택: 색상/사진배경) ──── 5-1-2 사진 배경 variant
    │ (공유하기)
    ▼
5-2 공유 시트 ──── (링크/인스타/메시지/저장)
    │ (저장 → 수집)
    ▼
6 문장 수집 (메모 + 인용문 + 태그)
```

## 4. 화면별 상세 기획

### 1 홈 (`a3p4g`)

**목적:** 앱 진입 시 오늘의 문장과 큐레이션 콘텐츠 노출

| 영역 | 내용 |
|---|---|
| 헤더 | Bookmate 로고 (Outfit 26/700) + 알림 벨 아이콘 |
| 날짜/타이틀 | "3월 19일 수요일" + "오늘의 문장" |
| Featured Quote 카드 | 큰따옴표(coral) + 인용문 + 출처 (책명·저자) / 카드 shadow |
| Curation Section | 섹션 헤더(더보기) + 가로 스크롤 카드 목록 |
| Recommend Section | "이런 문장은 어때요?" + 추천 문장 카드 |
| 탭바 | 홈 / (중앙) / 내 문장 — 3탭 pill 형태 |

**핵심 동작:**
- 홈 카드 탭 → 문장 상세 진입
- 탭바 중앙 버튼 → 스캔 플로우 시작 (도서 선택)

---

### 2 도서 선택 (`KjHwV`)

**목적:** 문장을 어느 책에서 수집할지 지정

| 영역 | 내용 |
|---|---|
| 헤더 | "도서 선택" + X(닫기) |
| 검색바 | "책 제목 또는 저자를 검색하세요" (search 아이콘) |
| Recent Section | 최근 선택한 도서 목록 |

**핵심 동작:**
- 검색 → 도서 선택 → 카메라 화면으로 이동
- X → 플로우 중단, 홈으로 복귀

---

### 3-1 사진 촬영 (`p67S4`)

**목적:** 책 페이지를 카메라로 촬영

| 영역 | 내용 |
|---|---|
| 상단 바 | X(닫기) / "문장 스캔" / 플래시(zap) — 흰 텍스트 |
| Book Info Bar | book-open 아이콘 + "책명 · 저자명" (반투명 흰색) |
| 뷰파인더 | 카메라 프리뷰 전체화면 + 코너 가이드라인 4개 (accent색) + 힌트 텍스트 |
| 하단 컨트롤 | 갤러리 버튼 / 셔터(흰 원테두리) / 텍스트 직접입력 모드(T) |

**핵심 동작:**
- 셔터 탭 → 3-2 사진 확인
- 갤러리 → 포토 라이브러리에서 선택
- T 버튼 → 문장 직접 입력 모드 (6 문장 수집으로 바로 진입)

---

### 3-2 사진 확인 (`Qd8np`)

**목적:** 촬영된 사진 품질 확인 후 진행 여부 결정

| 영역 | 내용 |
|---|---|
| 사진 영역 | 촬영 이미지 전체화면 표시 |
| 하단 버튼 | "다시 촬영" (아웃라인) / "텍스트 인식" (accent fill) |

**핵심 동작:**
- 다시 촬영 → 3-1로 복귀
- 텍스트 인식 → OCR 처리 후 3-3으로 이동

---

### 3-3 텍스트 인식 (`VRu1h`)

**목적:** OCR 인식 결과 확인 및 인식 설정 조정

| 영역 | 내용 |
|---|---|
| 헤더 | back chevron + "텍스트 인식" |
| 사진 섹션 | 사진 미리보기 + OCR 인식 영역 하이라이트 (accent 반투명 사각형) |
| 설정 카드 | 정확도(Accuracy) / 교정(Correction) / 언어(Language) 토글 rows |
| 하단 | "텍스트 확인하기" (accent 버튼) |

**핵심 동작:**
- 인식 영역 시각적으로 표시 (5개 라인 하이라이트)
- 설정 조정 후 "텍스트 확인하기" → 4 문장 선택으로 이동

### 4 문장 선택 (`ZZQXo`)

**목적:** 인식된 텍스트 중 수집할 문장의 줄을 직접 선택

| 영역 | 내용 |
|---|---|
| 헤더 | back + "문장 선택" / "최대 5줄" (subtitle) |
| 텍스트 목록 | 줄 단위 행 나열, tap으로 선택/해제 |
| 선택 상태 | 선택됨: accent fill + 흰 텍스트 / 미선택: 회색 텍스트 |
| 하단 | "계속" (accent 버튼) |

**핵심 동작:**

- Bookmate 로고 워터마크 (카드 하단)

**핵심 동작:**
- 스타일 스와치 탭 → 카드 배경 즉시 변경

## 5. 핵심 기능 정의

### 5-1 카메라 & OCR

- AVFoundation으로 카메라 세션 관리
- Vision Framework (`VNRecognizeTextRequest`)으로 한국어 OCR 수행
- 인식 영역을 라인 단위 bounding box로 표시
- 사용자가 줄(line) 단위로 선택 → 선택된 줄 합쳐서 문장 구성
- 최대 5줄 선택 제한

### 5-2 카드 생성

- 선택한 스타일(색상 5종 + 사진 배경)로 UIView 렌더링
- UIGraphicsImageRenderer로 카드 이미지 생성 (공유/저장용)
- 카드 구성: 문장 텍스트 + 출처 + Bookmate 워터마크

### 5-3 데이터 모델

```
Book
├── id: UUID
├── title: String
├── author: String
└── coverImage: Data?

Quote
├── id: UUID
├── text: String          // 수집된 문장
├── memo: String?         // 나의 메모
├── tags: [Tag]
├── book: Book
├── pageNumber: Int?
├── createdAt: Date
└── cardStyle: CardStyle  // 저장 당시 카드 스타일

Tag
├── id: UUID
└── name: String          // 자아, 성장, 사랑, 위로 등

CardStyle
└── type: Enum(green, coral, dark, white, blue, photo)
```

### 5-4 태그 시스템

- 사용자 정의 태그 자유 생성
- 추천 태그 제공 (앱 기본 제공)
- 복수 태그 동시 필터링 지원 (AND 조건)
- "미지정" 필터 = 태그 없는 문장

### 5-5 공유

- 카드 이미지 → 카메라롤 저장 (`PHPhotoLibrary`)
- 링크 복사, 인스타그램, 메시지 → `UIActivityViewController`

## 8. 미설계 영역 (추후 기획 필요)

- 문장 상세 보기 화면
- 알림(벨) 기능
- 큐레이션 콘텐츠 관리 (홈 피드)
- 검색 결과 화면 (내 문장 검색)
- 온보딩 / 권한 요청 (카메라, 사진 라이브러리)
- 설정 화면 (테마, 계정)
- 책 상세 정보 / 책별 문장 목록
